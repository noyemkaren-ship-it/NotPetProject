# Tratuar.ru — Сайт тротуарной плитки

Современный сайт-визитка с админ-панелью для компании по продаже тротуарной плитки и строительных материалов.

## Технологии

- **Backend**: Node.js + Express
- **База данных**: Prisma + SQLite
- **Авторизация**: JWT + Cookies
- **Загрузка файлов**: Multer
- **Логирование**: Pino
- **Шаблонизатор**: EJS
- **HTTPS**: Let's Encrypt

## Основные возможности

- Красивый адаптивный дизайн (главная + страница "О нас")
- Приём заявок от клиентов
- Админ-панель с CRUD-операциями:
  - Управление материалами
  - Управление услугами
  - Загрузка и управление фото
  - Просмотр и удаление заявок
- Защита от перегрузок (Rate Limit)
- Загрузка изображений (jpg, png, webp, gif)
